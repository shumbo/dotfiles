Dotfiles
===

## Usage

### MacOS

```bash
# bootstrap a machine from new, runs install, symlink, config
make bootstrap-macosx

# install applications
make install-macosx

# symlink configuration files
make symlink-macosx

# configure osx look and feel
make config-macosx
```
